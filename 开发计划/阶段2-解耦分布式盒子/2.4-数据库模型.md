# 阶段2：解耦分布式盒子 — 数据库模型

## 1. AiBoxModel — AI 推理盒子

```python
class AiBoxModel(MappedBase):
    __tablename__ = "video_ai_boxes"
    __table_args__ = ({'comment': 'AI推理盒子'})

    id: Mapped[int] = mapped_column(Integer, primary_key=True, autoincrement=True)
    name: Mapped[str] = mapped_column(String(64), nullable=False, comment="盒子名称")
    box_id: Mapped[str] = mapped_column(
        String(64), unique=True, nullable=False,
        comment="盒子唯一标识（硬件MAC/SN序列号）"
    )

    # 认证
    secret_key: Mapped[str] = mapped_column(
        String(128), nullable=False,
        comment="注册密钥（预共享，用于首次注册验证）"
    )
    access_token: Mapped[str | None] = mapped_column(
        String(512), nullable=True,
        comment="当前有效的 JWT token"
    )

    # 网络
    ip_address: Mapped[str | None] = mapped_column(String(64), nullable=True, comment="IP 地址")
    agent_port: Mapped[int] = mapped_column(Integer, default=9510, comment="Agent 监听端口（保留）")
    api_version: Mapped[str | None] = mapped_column(String(16), nullable=True, comment="Agent 版本")

    # 状态
    status: Mapped[str] = mapped_column(
        String(16), default="OFFLINE",
        comment="ONLINE / OFFLINE / ERROR"
    )
    last_heartbeat: Mapped[datetime | None] = mapped_column(
        DateTime, nullable=True, comment="最后心跳时间"
    )

    # 硬件信息
    gpu_info: Mapped[dict | None] = mapped_column(
        JSONB, nullable=True,
        comment="GPU 信息: {model, memory_mb, driver_version, cuda_version}"
    )
    cpu_info: Mapped[dict | None] = mapped_column(
        JSONB, nullable=True,
        comment="CPU 信息: {cores, arch}"
    )
    memory_total: Mapped[int | None] = mapped_column(
        Integer, nullable=True, comment="总内存（MB）"
    )

    # 资源使用（最新心跳上报）
    resource_usage: Mapped[dict | None] = mapped_column(
        JSONB, nullable=True,
        comment="资源使用率: {gpu_util, gpu_mem_used, cpu_percent, memory_used}"
    )

    # 容量
    max_tasks: Mapped[int] = mapped_column(Integer, default=4, comment="最大并发任务数")

    # 部署
    location: Mapped[str | None] = mapped_column(
        String(128), nullable=True, comment="部署位置描述"
    )
    description: Mapped[str | None] = mapped_column(Text, nullable=True, comment="备注")

    # 时间
    created_at: Mapped[datetime] = mapped_column(DateTime, default=datetime.now)
    updated_at: Mapped[datetime] = mapped_column(
        DateTime, default=datetime.now, onupdate=datetime.now
    )
```

## 2. AiBoxTaskModel — 盒子任务分配

```python
class AiBoxTaskModel(MappedBase):
    __tablename__ = "video_ai_box_tasks"
    __table_args__ = ({'comment': 'AI盒子任务分配'})

    id: Mapped[int] = mapped_column(Integer, primary_key=True, autoincrement=True)

    # 关联
    algorithm_task_id: Mapped[int] = mapped_column(
        Integer, ForeignKey("video_algorithm_tasks.id", ondelete="CASCADE"),
        unique=True, nullable=False,
        comment="算法任务 ID（一对一）"
    )
    box_id: Mapped[int] = mapped_column(
        Integer, ForeignKey("video_ai_boxes.id", ondelete="CASCADE"),
        nullable=False, index=True,
        comment="AI 盒子 ID"
    )

    # 状态
    local_status: Mapped[str] = mapped_column(
        String(16), default="PENDING",
        comment="PENDING / RUNNING / STOPPED / ERROR / UNKNOWN"
    )
    # PENDING: 等待 Agent 拾取
    # RUNNING: Agent 已启动 Worker
    # STOPPED: 已停止
    # ERROR: Worker 异常退出
    # UNKNOWN: 盒子断线，状态未知

    # 时间
    assigned_at: Mapped[datetime] = mapped_column(
        DateTime, default=datetime.now, comment="分配时间"
    )
    started_at: Mapped[datetime | None] = mapped_column(
        DateTime, nullable=True, comment="实际启动时间"
    )
    stopped_at: Mapped[datetime | None] = mapped_column(
        DateTime, nullable=True, comment="实际停止时间"
    )

    # 离线韧性：Agent 缓存的配置快照
    task_config: Mapped[dict] = mapped_column(
        JSONB, nullable=False,
        comment="任务配置快照（包含 Worker 所需的全部参数）"
    )

    # Agent 上次上报的指标
    last_reported_fps: Mapped[float | None] = mapped_column(
        Float, nullable=True, comment="最近上报的 FPS"
    )
    last_reported_uptime: Mapped[int | None] = mapped_column(
        Integer, nullable=True, comment="最近上报的运行时长（秒）"
    )
```

## 3. 索引

```sql
CREATE INDEX idx_ai_box_tasks_box_id ON video_ai_box_tasks(box_id);

CREATE INDEX idx_ai_boxes_status ON video_ai_boxes(status);
CREATE INDEX idx_ai_boxes_box_id ON video_ai_boxes(box_id);
```

## 4. 表关系

```
AiBoxModel (1) ────── (N) AiBoxTaskModel (1) ────── (1) AlgorithmTaskModel
     │                                                    │
     │                                                    │
     └── 盒子注册信息 ──────────────────────────────────┘  算法任务定义
```

## 5. 数据一致性

| 场景 | 处理 |
|------|------|
| 删除盒子 | 检查该盒子无 RUNNING 任务 |
| 删除算法任务 | CASCADE 删除关联的 AiBoxTaskModel |
| Agent 心跳超时 | 标记盒子 OFFLINE，任务 UNKNOWN |
| Agent 重新上线 | 同步任务状态，Web 端下发差异命令 |
