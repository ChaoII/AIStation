# 阶段2：解耦分布式盒子 — 前端管理页面

## 1. AI 盒子管理页

**文件：** `frontend/src/views/module_video/ai-box/index.vue`

标准 CRUD 页面（参考 `deploy/index.vue` 的 PageSearch + PageContent + EnhancedDialog 模式）。

### 搜索栏

| 字段 | 类型 | 说明 |
|------|------|------|
| 盒子名称 | Input | 模糊搜索 |
| 状态 | Select | ONLINE / OFFLINE / ERROR |
| IP | Input | 精确搜索 |

### 表格列

| 列 | 渲染 | 说明 |
|----|------|------|
| 名称 | 文本 | 自定义名称 |
| Box ID | 文本 | 硬件标识 |
| IP | 文本 | 最后上报 IP |
| GPU | tooltip | 型号 + 显存使用率 |
| 任务数 | 标签 | `2/4`（运行数/最大） |
| 状态 | el-tag | ONLINE=绿, OFFLINE=灰, ERROR=红 |
| 最后心跳 | 相对时间 | "30秒前" / "5分钟前" |
| 位置 | 文本 | 部署位置 |
| 操作 | 按钮 | 编辑 / 删除 |

### 配置磁盘

"GPU 信息"和"资源使用率"使用 Tooltip 展示详细数据：

```html
<el-tooltip placement="top">
  <template #content>
    <div>型号: Jetson Orin NX</div>
    <div>显存: 4096/8192 MB</div>
    <div>驱动: 12.2</div>
  </template>
  <span>Jetson Orin NX (50%)</span>
</el-tooltip>
```

### 编辑/新增表单

| 字段 | 类型 | 必填 |
|------|------|------|
| 名称 | Input | 是 |
| Box ID | Input | 是（不可修改） |
| 注册密钥 | Input | 是（新建时必填） |
| 最大任务数 | InputNumber | 是 |
| 位置 | Input | 否 |
| 描述 | Textarea | 否 |

## 2. 布控页面修改

### 2.1 表单增加"AI 盒子"字段

在现有的布控表单中增加：

```html
<el-form-item label="AI 盒子" prop="box_id">
  <el-select
    v-model="formData.box_id"
    filterable
    placeholder="选择 AI 盒子（可选）"
    style="width: 280px"
  >
    <el-option
      v-for="b in boxOptions"
      :key="b.id"
      :label="`${b.name} (${b.running_tasks}/${b.max_tasks})`"
      :value="b.id"
      :disabled="b.running_tasks >= b.max_tasks"
    />
  </el-select>
</el-form-item>
```

### 2.2 列表增加"运行盒子"列

```html
<el-table-column label="运行盒子" min-width="140">
  <template #default="scope">
    <el-tag v-if="scope.row.box_name" size="small">
      {{ scope.row.box_name }}
    </el-tag>
    <span v-else class="text-muted">未分配</span>
  </template>
</el-table-column>
```

### 2.3 修改启动流程

启动推理时：
- 如果 `box_id` 已分配 → 更新 `AiBoxTaskModel.status=PENDING`，等待 Agent 下次心跳拾取
- 如果未分配 → 按阶段1逻辑在本地启动（兼容模式）

### 2.4 状态显示增强

在列表状态列中，当任务运行在 AI 盒子上时，显示 "远程运行" 标识：

```html
<template #default="scope">
  <el-tag :type="statusTagType(scope.row)" size="small">
    {{ statusLabel(scope.row) }}
  </el-tag>
  <el-tag v-if="scope.row.box_name" type="info" size="small" effect="plain">
    {{ scope.row.box_name }}
  </el-tag>
</template>
```

## 3. 新建 API 文件

```typescript
// frontend/src/api/module_video/ai-box.ts
import request from "@/utils/request";

export function getAiBoxList(data?: any) {
  return request({ url: "/video/algorithm/box/list", method: "get", params: data });
}

export function createAiBox(data: any) {
  return request({ url: "/video/algorithm/box/create", method: "post", data });
}

export function updateAiBox(id: number, data: any) {
  return request({ url: `/video/algorithm/box/update/${id}`, method: "put", data });
}

export function deleteAiBox(ids: number[]) {
  return request({ url: "/video/algorithm/box/delete", method: "delete", data: { ids } });
}

export function getAiBoxDetail(id: number) {
  return request({ url: `/video/algorithm/box/detail/${id}`, method: "get" });
}
```
