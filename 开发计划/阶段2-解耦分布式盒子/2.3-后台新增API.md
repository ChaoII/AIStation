# 阶段2：解耦分布式盒子 — 后台新增 API

## 1. AI 盒子注册

```
POST /api/v1/algorithm/agent/register
```

**公开接口**（不依赖用户 JWT，依赖 `secret_key` 验证）。

**请求体：**
```json
{
  "box_id": "AI-BOX-001A",
  "name": "门口AI盒",
  "secret_key": "预共享密钥",
  "ip_address": "10.0.1.50",
  "agent_port": 9510,
  "api_version": "1.0.0",
  "gpu_info": {"model": "Jetson Orin NX", "memory_mb": 8192},
  "cpu_info": {"cores": 8},
  "memory_total": 16384,
  "max_tasks": 4
}
```

**处理逻辑：**
1. 根据 `box_id` 查 `AiBoxModel`
2. 验证 `secret_key` 匹配
3. 如不存在且密钥有效则创建新记录
4. 生成短期 JWT token 返回（用于后续心跳认证）
5. 更新 `status=ONLINE`、`last_heartbeat=now`

**响应：**
```json
{
  "code": 200,
  "data": {
    "box_db_id": 1,
    "token": "eyJhbGciOiJIUzI1NiIs..."
  },
  "msg": "注册成功"
}
```

## 2. 心跳 + 命令轮询

```
POST /api/v1/algorithm/agent/heartbeat
```

**认证：** `Authorization: Bearer {register下发的JWT}`

**请求体：**
```json
{
  "box_id": 1,
  "token": "jwt...",
  "timestamp": "2026-05-31T10:00:00Z",
  "tasks": [
    {"task_id": 1, "local_status": "RUNNING", "fps": 12.5, "uptime_sec": 3600},
    {"task_id": 2, "local_status": "RUNNING", "fps": 8.3, "uptime_sec": 1800}
  ],
  "resource_usage": {
    "gpu_util_percent": 65,
    "gpu_mem_used_mb": 4096,
    "cpu_percent": 45,
    "memory_used_mb": 8192
  }
}
```

**处理逻辑：**
1. 验证 JWT
2. 更新 `AiBoxModel.last_heartbeat=now`, `status=ONLINE`
3. 更新 `resource_usage` 字段
4. 同步任务状态：Agent 上报的 `local_status` 同步到 `AiBoxTaskModel.local_status`
5. 查询 `AiBoxTaskModel` 中该盒子下 `status=PENDING` 或 `status=STOPPED_WEB` 的任务
6. 组装命令列表返回

**命令响应：**

## 3. AI 盒子 CRUD

标准管理接口（复用 `AlgorithmRouter`，使用 `module_video:algorithm` 权限前缀）：

| 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|
| GET | `/algorithm/box/list` | query | 分页查询盒子列表 |
| GET | `/algorithm/box/detail/{id}` | query | 盒子详情（含运行任务列表） |
| POST | `/algorithm/box/create` | create | 手动添加盒子 |
| PUT | `/algorithm/box/update/{id}` | update | 编辑盒子信息 |
| DELETE | `/algorithm/box/delete` | delete | 删除盒子（需确保无运行任务） |

## 4. 布控任务分配盒子

修改 `POST /algorithm/task/create` 和 `PUT /algorithm/task/update/{id}`：

新增可选字段 `box_id`：
```json
{
  "camera_id": 24,
  "algorithm_id": 1,
  "box_id": 1,
  ...
}
```

处理逻辑：
- 创建时若指定 `box_id`，同时插入 `AiBoxTaskModel`
- 更新时若 `box_id` 变更，更新 `AiBoxTaskModel`
- 启动推理时（start），由 Agent 的心跳轮询拾取任务

## 5. 心跳超时检测（scheduler）

```python
async def check_box_heartbeats():
    """每 60s 检查所有 ONLINE 盒子的最后心跳时间"""
    while True:
        await asyncio.sleep(60)
        async with async_db_session() as session:
            result = await session.execute(
                select(AiBoxModel).where(AiBoxModel.status == "ONLINE")
            )
            for box in result.scalars():
                if not box.last_heartbeat:
                    continue
                elapsed = (datetime.now() - box.last_heartbeat).total_seconds()
                if elapsed > 180:  # 3 次心跳超时
                    box.status = "OFFLINE"
                    # 相关任务标记为 UNKNOWN
                    await mark_tasks_unknown(box.id, session)
```
