## 1. Prometheus 指标

### 1.1 Web 主站指标

```python
from prometheus_client import Counter, Gauge, Histogram

# API 请求
api_request_total = Counter("api_request_total", "API 总请求数", ["method", "endpoint", "status"])
api_request_duration = Histogram("api_request_duration_seconds", "API 请求耗时", ["method", "endpoint"])

# 推理任务
inference_tasks_running = Gauge("inference_tasks_running", "当前运行中的推理任务数")
inference_tasks_total = Counter("inference_tasks_total", "推理任务总数")
inference_events_total = Counter("inference_events_total", "检测事件总数", ["algorithm_type"])
inference_latency = Histogram("inference_latency_seconds", "推理延迟", ["algorithm_type"])

# 告警
alarm_total = Counter("alarm_total", "告警总数", ["alarm_type", "severity"])
alarm_ws_connections = Gauge("alarm_ws_connections", "WebSocket 连接数")

# AI 盒子
ai_boxes_online = Gauge("ai_boxes_online", "在线盒子数")
ai_boxes_offline = Gauge("ai_boxes_offline", "离线盒子数")

# Worker 回调
callback_total = Counter("callback_total", "Worker 回调总数", ["status"])
callback_duration = Histogram("callback_duration_seconds", "回调处理耗时")
```

### 1.2 AI 盒子指标

Agent 暴露 `/metrics` 端点：

```python
# 盒子侧指标
worker_running = Gauge("worker_running", "本地 Worker 运行数")

```

## 3. 日志聚合

### 3.1 Worker 日志

Worker 输出 JSON 到 stderr，格式：

```json
{"t":"2026-05-31T10:00:00Z","lvl":"INFO","task":1,"msg":"heartbeat","fps":12.5,"latency_ms":8.3}
{"t":"2026-05-31T10:00:05Z","lvl":"DETECT","task":1,"camera":24,"detections":1,"labels":["person"]}
{"t":"2026-05-31T10:00:06Z","lvl":"CALLBACK","task":1,"status":200,"duration_ms":45}
```

### 3.2 Agent 日志

Agent 输出 JSON 到文件 + stdout：

```json
{"t":"...","lvl":"INFO","msg":"heartbeat_success","latency_ms":120,"commands":1}
{"t":"...","lvl":"WARN","msg":"web_unreachable","retry_attempt":3}
{"t":"...","lvl":"INFO","msg":"task_started","task_id":1,"pid":1234}
```

## 4. 告警规则

```yaml
# prometheus-rules.yaml
groups:
  - name: ai-box
    rules:
      - alert: BoxOffline
        expr: ai_boxes_offline > 0
        for: 5m
        labels: { severity: critical }
        annotations: { summary: "AI 盒子离线" }

      - alert: InferenceLatencyHigh
        expr: inference_latency_seconds{quantile="0.95"} > 0.5
        for: 5m
        labels: { severity: warning }
        annotations: { summary: "推理延迟过高" }

      - alert: GpuUtilHigh
        expr: gpu_util_percent > 90
        for: 10m
        labels: { severity: warning }
        annotations: { summary: "GPU 利用率过高" }

      - alert: WorkerCrashed
        expr: inference_tasks_running - worker_running > 0
        for: 1m
        labels: { severity: critical }
        annotations: { summary: "有 Worker 异常退出" }
```

## 6. 运维命令

```bash
# 列出所有盒子和任务
curl /api/v1/algorithm/box/list?page_size=100

# 查看某个盒子的详情和运行任务
curl /api/v1/algorithm/box/detail/1

# 强制迁移任务
curl -X POST /api/v1/algorithm/task/5/migrate -d '{"to_box_id": 2}'

# 盒子维护模式（不再分配新任务）
curl -X PUT /api/v1/algorithm/box/1/maintenance

# 查看 Worker 实时日志
ssh ai-box-1 "journalctl -u ai-box-agent -f --since '5 min ago'"
```
