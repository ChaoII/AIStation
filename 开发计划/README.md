# FastapiAdmin × ModelDeploy 推理引擎集成 — 开发计划

## 项目目标

将 ModelDeploy C++ 推理引擎集成到 FastapiAdmin 视频监控系统，实现 **摄像头实时流 AI 分析 → 事件告警 → 前端可视化** 的完整链路。

## 三阶段实施路线

```
第一阶段 ──── 第二阶段 ──── 第三阶段
 单机验证      分布式盒子     规模化运营
  ┌─────┐      ┌─────┐       ┌─────┐
  │ Web  │      │ Web │       │ Web │
  │ +   │ ──▶  │  ↑  │ ──▶  │HA↑  │
  │推理  │      │Agent│       │集群 │
  └─────┘      └─────┘       └─────┘
  快速验证     解耦拆分       生产就绪
```
