### 1.2 状态列增强

当前状态列显示 RUNNING/STOPPED 标签，改为实时状态：

| 状态 | 显示 | 说明 |
|------|------|------|
| RUNNING | 绿色标签 "运行中" + FPS 数值 | 通过 `getInferenceStatus` 轮询更新 |
| STOPPED | 灰色标签 "已停止" | - |
| ERROR | 红色标签 "异常" | 鼠标悬停显示错误信息 |
| UNKNOWN | 黄色标签 "未知" | Agent 断线等情况 |

### 1.3 启动/停止逻辑

```typescript
async function handleToggleInference(row: any) {
  if (row.status === 'RUNNING') {
    await stopInferenceTask(row.id);
    ElMessage.success('推理已停止');
  } else {
    await startInferenceTask(row.id);
    ElMessage.success('推理已启动');
  }
  refreshList();
}

// 轮询在线状态（仅对 RUNNING 任务）
onMounted(() => {
  statusTimer = setInterval(async () => {
    const runningTasks = tableData.filter(t => t.status === 'RUNNING');
    for (const task of runningTasks) {
      const res = await getInferenceStatus(task.id);
      task.inferenceStatus = res.data.data;
    }
  }, 5000);  // 每 5s 刷新
});

onBeforeUnmount(() => {
  clearInterval(statusTimer);
});
```

## 2. DetectionOverlay 组件

**文件：** `frontend/src/components/Video/DetectionOverlay.vue`

### 2.1 组件接口

```typescript
interface Detection {
  label: string
  confidence: number
  bbox: { x: number; y: number; width: number; height: number }
}

interface Props {
  detections: Detection[]
  detectRegion?: number[][]    // 归一化多边形
  videoWidth: number
  videoHeight: number
}
```

### 2.2 渲染逻辑

```html
<template>
  <canvas
    ref="canvasRef"
    class="detection-overlay"
    :width="videoWidth"
    :height="videoHeight"
    style="pointer-events: none; position: absolute; top: 0; left: 0;"
  />
</template>
```

每次 detections 变化时：
1. `ctx.clearRect(0, 0, w, h)`
2. 画 ROI 多边形（半透明填充 + 描边）
3. 遍历 detections：
   - 矩形：`ctx.strokeRect`，线宽 2px
   - 颜色：按 label 分配（`colorMap[label]`）
   - 标签：`ctx.fillText` 在矩形上方（label + confidence%）
   - 字体：12px sans-serif，白色背景半透明

## 3. WebSocket 集成

### 3.1 在 LivePlayer.vue 中

```typescript
const detections = ref<Detection[]>([])

// 建立 WS 连接（复用现有 AlarmWSManager）
const wsUrl = `${location.protocol === 'https:' ? 'wss:' : 'ws:'}//${apiBase}/api/v1/alarm/ws?token=${getToken()}`
const ws = new WebSocket(wsUrl)

ws.onmessage = (e) => {
  const msg = JSON.parse(e.data)
  // 只处理当前播放相机的告警
  if (msg.camera_id === props.cameraId && msg.detections) {
    detections.value = msg.detections
    // 3s 后自动清除
    setTimeout(() => { detections.value = [] }, 3000)
  }
}
```

### 3.2 模板集成

```html
<div class="video-wrapper" style="position: relative;">
  <video ... />
  <DetectionOverlay
    v-if="showDetection && detections.length > 0"
    :detections="detections"
    :video-width="videoWidth"
    :video-height="videoHeight"
  />
</div>
```
