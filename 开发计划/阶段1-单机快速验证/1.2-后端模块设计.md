## 2. worker.py — 推理 Worker

**启动方式**：`python worker.py --config /tmp/infer_{task_id}.json`

**Config JSON 完整结构：**
```json
{
  "task_id": 1,
  "camera_id": 24,
  "algorithm_type": "INTRUSION",
  "stream_url": "http://127.0.0.1:80/live/camera_24.live.flv",
  "model_path": "/data/models/intrusion/engine.onnx",
  "runtime_config": {
    "engine": "tensorrt",
    "gpu": {"enabled": true, "device_id": 0},
    "threads": 4,
    "batch_size": 1,
    "input_width": 640,
    "input_height": 640
  },
  "preset_params": {
    "conf_threshold": 0.5,
    "nms_threshold": 0.45
  },
  "detect_region": [[0.1,0.1],[0.9,0.1],[0.9,0.9],[0.1,0.9]],
  "callback_url": "http://127.0.0.1:8001/api/v1/algorithm/detection/callback",
  "callback_token": "infer_shared_secret_token",
  "fps_target": 5,
  "alarm_interval": 30,
  "snapshot_dir": "data/detections/"
}
```

**Worker 主循环逻辑：**

```
初始化:
  load_config() → 读取 JSON
  create_model(config) → 初始化 ModelDeploy 模型
  cap = VideoCapture(config.stream_url) → 打开流
  register_signal_handler(SIGTERM/SIGINT) → 设置 stop_flag

主循环:
  while not stop_flag:
    ret, frame = cap.read()
    if not ret:
      sleep(1); reconnect() → continue
    
    frame_count++
    if frame_count % skip_frames < 1:
      if detect_region:
        frame = apply_roi_mask(frame, detect_region)
      
      # 推理
      t0 = time()
      results = model.predict(frame)
      latency = time() - t0
      
      # 过滤
      alarms = []
      for r in results:
        if r.score < conf_threshold: continue
        if throttled(r.label, alarm_interval): continue
        alarms.append(r)
      
      if alarms:
        # 标注快照
        annotated = draw_boxes(frame, alarms)
        snapshot_path = save_jpeg(annotated, snapshot_dir)
        
        # 回调
        http_post(callback_url, {
          "task_id": task_id,
          "camera_id": camera_id,
          "algorithm_type": algorithm_type,
          "detections": [r.to_dict() for r in alarms],
          "snapshot_data": base64(annotated),
          "snapshot_path": snapshot_path,
          "frame_timestamp": now().isoformat(),
          "inference_latency_ms": round(latency*1000, 1),
        })
    
    # 心跳
    if frame_count % 900 == 0:
      stderr_json({"type": "heartbeat", "task_id": task_id, "fps": ...})
  
  清理:
    cap.release()
    stderr_json({"type": "shutdown", "task_id": task_id, ...})
```

**关键设计：**
- 使用 `sys.stderr` 输出 JSON 日志（stdout 保留给 print 调试）
- 所有 HTTP 请求使用 `httpx.AsyncClient` 或同步 `requests`（Worker 不依赖 asyncio）
- 快照 JPEG 质量 85%，分辨率 1280×720
- 断流自动重连：`cap.release()` → sleep(5) → `VideoCapture()`

## 4. schema.py — Pydantic 模型

```python
class DetectionEventSchema(BaseModel):
    task_id: int
    camera_id: int
    algorithm_type: str
    detections: list[DetectionItem]
    snapshot_data: str | None = None   # base64 JPEG
    snapshot_path: str | None = None
    frame_timestamp: datetime
    inference_latency_ms: float | None = None

class DetectionItem(BaseModel):
    label: str
    label_id: int
    confidence: float
    bbox: BBoxModel  # {x, y, width, height} 归一化坐标

class BBoxModel(BaseModel):
    x: float
    y: float
    width: float
    height: float

class InferenceStatusSchema(BaseModel):
    task_id: int
    status: str          # RUNNING/STOPPED/ERROR/UNKNOWN
    pid: int | None = None
    uptime_seconds: float | None = None
    fps: float | None = None

class InferenceConfigSchema(BaseModel):
    stream_url: str
    model_path: str
    algorithm_type: str
    runtime_config: dict = {}
    preset_params: dict = {}
    detect_region: list | None = None
    fps_target: int = 5
    alarm_interval: int = 30
```
